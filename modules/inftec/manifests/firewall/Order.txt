Rules are ordered by the comment. We use the following pattern:

* Pre-Rules: 000 - 099, pre.pp
* Post-Rules: 900 - 999, post.pp

* Application default rules: 100-149, default.pp
* Master server rules (Puppet, ...): 150-199, /nodes.pp

* Apache rules (http, https): 200-209, ../apache.pp
* DNS rules (http, https): 210-219, ../dns.pp
